### Recommended Specifications:
- **Resolution:** 800x800 pixels (square)
- **Aspect Ratio:** 1:1 (square) or 4:3
- **Format:** JPEG or PNG
- **File size:** Under 500KB (compress before uploading)

### Tips:
1. Use high-quality product photos
2. Ensure good lighting
3. Use a clean, simple background
4. Keep images consistent across all products
5. Compress images to improve page loading speed

### Naming Convention:
Images are automatically renamed when uploaded:
```
YYYYMMDDHHMMSS_originalname.ext
```

Example: `20241016084530_matcha_latte.jpg`
